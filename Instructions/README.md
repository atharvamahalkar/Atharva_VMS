# Architectural Ledger — VMS Portal
## AI-Powered Vendor Management System  |  Pure Java Application

**GitHub:** https://github.com/pvsh1804/Vendor-Managment-using-Java

## Tech Stack
| Component | Technology |
|---|---|
| Language | Java (JDK 11+) — entire application |
| LLM | Qwen:0.5B via Ollama (HTTP from Java) |
| Embeddings | nomic-embed-text via Ollama (HTTP from Java) |
| Vector DB | Pinecone (REST API called from Java) |
| Relational DB | PostgreSQL 15 (JDBC) |
| Frontend | HTML5 / CSS3 / JavaScript |
| AI Calls | Java HttpURLConnection |

## Quick Start
```bash
# 1. Start Ollama
ollama serve

# 2. Start Java app (run in Terminal form the base folder)

D:\ProjectFolder\ArchitecturalLedger> mvn spring-boot:run
# Portal: http://localhost:8080
```
